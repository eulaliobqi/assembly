# Pipeline de Montagem de Transcriptoma De Novo — *Mahanarva* sp.

## Glândula Salivar | RNA-Seq | Illumina Paired-End

---

## Descrição do Projeto

Pipeline completo para montagem de transcriptoma *de novo* a partir de dados de
RNA-Seq de glândula salivar do inseto *Mahanarva* sp. (Hemiptera: Cercopidae),
conhecido popularmente como cigarrinha-das-pastagens.

Os dados são provenientes de sequenciamento de transcritos (cDNA), e o objetivo
final é a obtenção de um transcriptoma de novo utilizando o Trinity v2.15.2.
